# {{PROJECT_NAME}} — Skills

Skills are step-by-step recipes for common development tasks. Follow them exactly to produce spec-compliant, documented, tested output every time.

## Difference: Roles vs Skills

- **Roles** = *"What conventions do I follow?"* (reference material)
- **Skills** = *"How do I do this specific task?"* (step-by-step recipe)
